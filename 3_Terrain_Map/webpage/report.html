Elevation mapping of Kitchener

This project consider elevation data for Kitchener, Ontario. A purpose of this project is to create 3D representation of the city of Kitchener. This report also looks at the cycling infrastructure in the city and how the elevation data can be used along cycling routes.

Geographic Information Systems project display by Mukto Akash.

Geopspatial skills and tools used.

The elevation levels of the Kitchener-Waterloo Region were obtained from the hillshade surface model from the Kitchener GeoHub. The elevation data was processed using the QGIS raster tools and used to create a hillshade and displayed in 3D using the QGIS 3D map view. The z-factor was exaggerated by a multiple of 5 to enhance the 3D effect. A greenish hue was used to represent the elevation levels, which gives a false sense of vegetation in the region.

A second representation of the elevation data was created using the digital elevation model data from the Kitchener Geohub. A similar process was used to create 3D representation of the elevation data. A layer of land cover data was used to overlay the elevation data to give a more realistic representation of the region. Data used for the project and how they were transformed.

All data was obtained from Kitchener GeoHub and modified to suit the requirements of the project. The data was transformed using QGIS processing tools and the attribute tables.

Analysis and results of the project.

A possible application of the elevation data would be to investigate bike routes in the city of Kitchener. The elevation data can be used to identify the steepest routes in the city and provide a more challenging route for cyclists. The elevation data can also be used to identify the most accessible routes for cyclists. In the map below, a layer of routes is superimposed on the elevation data to show how the routes are affected by the elevation levels. The steeper routes are shown in red, while the flatter routes are shown in greenish-blue. Bike routes are indicated in bright-red. Building footprints are shown in greyish purple. There are also contour lines to show the details of the change in elevation levels.

Challenges faced with the project.

The main challenge with the project was in obtaining reliable raster data to depict elevation in the city of Kitchener. Although there is some elevation data in the Kitchener GeoHub, the hope to obtained more detailed satellite imagery to show the city in more detail. Unfortunately, the aerial imagery data was too large to process into a 3D model.
