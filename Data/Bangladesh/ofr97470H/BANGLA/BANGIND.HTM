DATA DIRECTORY                  bangla

                                      ArcView 3.2 shape files.  There are five (or three)  files
                                      for every ArcView theme:
  .shp                           - the file that stores the feature geometry.
  .shx                           - the file that stores the index of the feature geometry.
  .dbf                           - the dBASE file that stores the attribute information of features. 
  .sbn and  .sbx,          -  the files that store the spatial index of the features.

                                    There are following shape files in bangla directory:

  abang1, mbang1, tbang1, sbang 1  - Undiscovered gas resource assessment themes.
                                                              Assessment units, Minimum Petroleum System,
                                                              Total Petroleum System, Pod of active Source Rocks
                                                               correspondingly.
  b_adming   - Administrative Boundary
  aerop          - Aeroports, Aerofields
  b_cityg       - Major Cities
  b_fltg          - Faults and Tectonic Contacts
  b_frameg    - Outer Frame
  b_geog       - Surface Geology
  b_gravg      - Bouguer Gravity Anomaly Contours
  b_magng     - Aeromagnetic Anomaly Contours
  b_rivg         - Rivers, Streams
  b_shoreg     - Sea shore Line
 bathym       - Bathymetric Contours
  ctry95g        - Country Boundary
  elevg           - Eleveation Contours
  grav_txt      - Artificial coverage to keep Hot Links to images with  text from original paper 
                       map with Bouguer Gravity Contours
  mag_txt       - Same for Aeromagnetic Anomaly Map
  Ref_geo      - Same for Geological Map
  lakes            - Closed Water Bodies
  latlong          - Graticular grid with 1 * 1 degree spacing
  physio_g       - Generalized Physiographic Map
  roads            - Roads
  Sec_ln          - Cross Section Lines
  tectong         - Generalized Tectonic Map
  water            - Water-Land coverage 

      * .avl        - All the ArcView Legend files.  They keep color, pattern to shade
                         polygons, draw lines and post markers.

           images    - Directory with text images from original map

           plot        - Directory with ready to plot files in EPS and PDF format.
