GIS Projects

This portfolio demonstrates some interesting data analysis performed on Geospatial data using various tools of Geographic Information Systems. The primary tools used are QGIS, python, MS-Excel, and Google Earth Engine. The projects are based on real-world data and are aimed at solving real-world problems.

Geographic Information Systems project display by Mukto Akash.
3D Maps

Some projects are visualized with the help of 3-dimensional maps that provide a better perspective of the terrain.

Classification

Machine Learning Algorithms such as kmeans classification on NVDI of raster data can be used to identify terrain.

Planning Routes

Various plugins can be used to enhance map functionalities. For example, the ORS plugin can be used to plot the shortest path between different points.
