Emergency Medical Services in the KW Region

This project studies the location of Emergency Medical Services such as hospitals, Urgent Care centers, Walk-in clinics and Emergency Ambulance services. Voronoi polygons are used to identify the regions closest to these facilities. The project also studies the distribution of these services in the KW region and identifies areas that are underserved.

Geographic Information Systems project display by Mukto Akash.

Geopspatial skills and tools used.

The base layer consists of an open street map. Emergency medical services such as hospitals, Urgent Care centers, Walk-in clinics and Emergency Ambulance services have been isolated and are displayed as points on the map, with topo hospital symbology. The Voronoi polygons are created using the QGIS processing tools and are displayed as transparent polygons with red borders.

A layer showing the Planning Communities of the Kitchener-Waterloo Region is superimposed with a semi-transparent layer to show the demarkation of the localities. This helps in identifying the distribution of the Emergency Medical Services in the region.

Finally, a heatmap is generated based on the estimated number of emergency room physicians in attendance at each facility. The heatmap is displayed with a colour ramp from green to red, with green indicating a higher number of physicians and red indicating a lower number of physicians. This layer is also used to label the facilities.

The QGIS tools used include "voronoi polygons" from vector geometry, and "clip vector by mask layer". A webmap was created using the QGIS2WEB plugin for interactive interface.

Data used for the project and how they were transformed.

All data, other than the open street map, was obtained from Kitchener GeoHub and modified to suit the requirements of the project. The data was transformed using QGIS processing tools and the attribute tables.
